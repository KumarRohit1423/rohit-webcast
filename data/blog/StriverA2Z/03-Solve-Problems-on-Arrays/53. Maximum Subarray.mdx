# [53. Maximum Subarray](https://leetcode.com/problems/maximum-subarray/description/)

Given an integer array `nums`, find the subarray with the largest sum, and return its sum.

**Example 1:**

```
Input: nums = [-2,1,-3,4,-1,2,1,-5,4]
Output: 6
Explanation: The subarray [4,-1,2,1] has the largest sum 6.
```

**Example 2:**

```
Input: nums = [1]
Output: 1
Explanation: The subarray [1] has the largest sum 1.
```

**Example 3:**

```
Input: nums = [5,4,-1,7,8]
Output: 23
Explanation: The subarray [5,4,-1,7,8] has the largest sum 23.
```

**Constraints:**

- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`

**Follow up:** If you have figured out the `O(n)` solution, try coding another solution using the **divide and conquer** approach, which is more subtle.

---

# My Approach

```cpp
class Solution {
 public:
  int maxSubArray(vector<int>& nums) {
    int sum = 0, bestSum = INT_MIN;
    for (int i = 0; i < nums.size(); i++) {
      sum = max(nums.at(i), (sum + nums.at(i)));
      bestSum = max(sum, bestSum);
    }
    return bestSum;
  }
};
```

# Notes

The `O(n)` solution implemented above is the [Kadane's Algorithm](https://en.wikipedia.org/wiki/Maximum_subarray_problem) .
We scan the array keeping a current sum value which is equal to prefix sum at that index. Keeping the maximum of all these gives us our answer.
