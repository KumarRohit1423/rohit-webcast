# [48. Rotate Image](https://leetcode.com/problems/rotate-image/description/)

You are given an `n x n` 2D `matrix` representing an image, rotate the image by **90** degrees (clockwise).

You have to rotate the image [in-place](https://en.wikipedia.org/wiki/In-place_algorithm), which means you have to modify the input 2D matrix directly. **DO NOT** allocate another 2D matrix and do the rotation.

**Example 1:**

<Image src="https://assets.leetcode.com/uploads/2020/08/28/mat1.jpg" width={500} height={188} />

```
Input: matrix = [[1,2,3],[4,5,6],[7,8,9]]
Output: [[7,4,1],[8,5,2],[9,6,3]]
```

**Example 2:**

<Image src="https://assets.leetcode.com/uploads/2020/08/28/mat2.jpg" width={500} height={201} />

```
Input: matrix = [[5,1,9,11],[2,4,8,10],[13,3,6,7],[15,14,12,16]]
Output: [[15,13,2,5],[14,3,4,1],[12,6,8,9],[16,7,10,11]]
```

**Constraints:**

- `n == matrix.length == matrix[i].length`
- `1 <= n <= 20`
- `-1000 <= matrix[i][j] <= 1000`

---

# My Approach

```cpp
class Solution {
 public:
  void rotate(vector<vector<int>>& matrix) {
    int a, b, n = matrix.size();
    for (int i = 0; i < (n / 2); i++) {
      for (int j = 0; j < (n - 1) - (2 * i); j++) {
        swap(matrix[i][j + i], matrix[j + i][matrix.size() - 1 - i]);
        swap(matrix[(matrix.size() - 1) - j - i][i], matrix[i][j + i]);
        swap(matrix[(matrix.size() - 1 - i)][(matrix.size() - 1) - j - i],
             matrix[(matrix.size() - 1) - j - i][i]);
      }
    }
  }
};
```

# Notes

Just learn how to swap rows (be it counter-clockwise or clockwise).
